# **App Name**: Finanzas Personales

## Core Features:

- Registro de Ingresos: Permite al usuario registrar sus ingresos mensuales especificando la fuente y cantidad.
- Registro de Gastos: Permite al usuario registrar sus gastos mensuales clasificándolos por categorías (ej. vivienda, comida, transporte).
- Visualización de Presupuesto: Muestra gráficos y tablas que resumen los ingresos y gastos del usuario por período, con un desglose por categorías.
- Alertas de Presupuesto: Genera alertas cuando el usuario supera los límites de gasto definidos para una categoría específica.
- Sugerencias de Ahorro: Ofrece sugerencias personalizadas de ahorro basadas en el análisis de los patrones de gasto del usuario, using a tool to evaluate where savings may be possible.
- Almacenamiento de datos: Utilizar firestore para almacenar datos
- Personalización: El usuario podrá elegir diferentes categorías de ingresos y egresos, con posibilidad de añadir nuevas

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to inspire confidence and stability in financial management.
- Background color: Very light blue (#E3F2FD), almost white, creating a clean and trustworthy backdrop.
- Accent color: Soft green (#81C784) to highlight savings and positive financial indicators.
- Body and headline font: 'PT Sans', a versatile humanist sans-serif for readability and a modern feel.
- Use clear, minimalist icons for categories like 'Food,' 'Transport,' and 'Entertainment' to enhance user understanding.
- Employ a clean and organized layout with clear sections for income, expenses, and budget overview.
- Use subtle transitions and animations when navigating between sections or displaying data updates for a smooth user experience.
